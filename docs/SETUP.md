# Setup Guide

## Prerequisites
- Node.js v18+
- MongoDB Atlas account
- Resend account

## Installation Steps

1. Install dependencies:
```bash
cd backend && npm install
```

2. Configure environment:
```bash
cp .env.example .env
# Edit .env with your credentials
```

3. Run locally:
```bash
npm run dev
```

4. Access:
- Backend: http://localhost:3000
- Frontend: Open frontend/index.html
